# Reverse String

## Problem Statement

Write a function that reverses a string.

## Examples

```
reverseString("hello") // "olleh"
reverseString("world") // "dlrow"
```

## Difficulty
Easy

## Topics
- String
- Two pointers

## Approach
Use a two-pointer technique to swap characters from both ends of the string until the middle is reached.

**Time Complexity:** O(n)
**Space Complexity:** O(n) (due to string to array conversion)
