# Sum Two Integers

Complete the function to compute the sum of two integers.

Example

Input: a = 2, b = 3

Output: 5

Function Description

Complete the sum function with the following parameters:

- a: the first value
- b: the second value

Returns

- int: the sum of a and b

Constraints

- -10^9 <= a, b <= 10^9

Sample Input

a = 2

b = 3

Sample Output

5
