# Hello, World!

## Problem Statement

Write a Python program that prints the exact text:

```
Hello, world!
```

The output must match capitalization and punctuation exactly.

## Example

**Program:**
```
print("Hello, world!")
```

**Output:**
```
Hello, world!
```

## Difficulty
Easy

## Topics
- Python

## Approach
Use a single `print()` statement to output the required phrase.
